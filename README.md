# GEOточка — фото эскиза в DXF

Рабочее локальное веб-приложение для преобразования фотографии геодезического эскиза в редактируемый DXF, который открывается в AutoCAD.

## Возможности

- загрузка JPEG/PNG до 20 МБ;
- автоматическая коррекция перспективы, когда видны границы листа;
- подавление теней и выравнивание контраста;
- поиск, привязка и объединение основных линий;
- визуальный редактор: добавление, удаление, изменение координат и перетаскивание концов;
- отмена/повтор действий;
- калибровка масштаба по одной линии известной длины;
- слои DXF `WALLS`, `OPENINGS`, `DIMENSIONS`, `REFERENCE`;
- работа без API-ключей и без сохранения загруженных фотографий.

> Автоматическое распознавание не заменяет проверку специалистом. Рукописные размеры и смысл элементов могут быть неоднозначны, поэтому перед использованием результата на объекте геометрию нужно проверить.

## Запуск на Windows

1. Установите [Python 3.11 или новее](https://www.python.org/downloads/). При установке отметьте **Add Python to PATH**.
2. Скачайте и распакуйте проект.
3. Дважды нажмите `start.bat`.
4. Откроется <http://127.0.0.1:8000>.

При первом старте установка библиотек может занять несколько минут.

## Запуск на macOS / Linux

```bash
chmod +x start.sh
./start.sh
```

Либо вручную:

```bash
python3 -m venv .venv
. .venv/bin/activate
pip install -e .
uvicorn app.main:app --host 127.0.0.1 --port 8000
```

## Запуск в Docker

```bash
docker compose up --build
```

Откройте <http://127.0.0.1:8000>.

## Как получить правильный DXF

1. Сфотографируйте лист целиком, без бликов, по возможности сверху.
2. Загрузите фотографию и дождитесь распознавания.
3. Удалите штрихи, относящиеся к цифрам и заметкам, а не к стенам.
4. Добавьте пропущенные линии инструментом **Линия**. Удерживайте `Shift` для горизонтали/вертикали.
5. Выберите отрезок с известной длиной, укажите длину в миллиметрах и нажмите **Рассчитать масштаб**.
6. Назначьте типы линий и нажмите **Скачать DXF**.
7. Откройте DXF в AutoCAD и при необходимости сохраните как DWG.

## Горячие клавиши

- `V` — выбор;
- `L` — новая линия;
- `Delete` — удалить выбранную линию;
- `Ctrl+Z` / `Ctrl+Y` — отменить / повторить;
- `Esc` — снять выбор.

## API

Интерактивная документация: <http://127.0.0.1:8000/docs>.

- `POST /api/v1/detect` — обработать JPEG/PNG;
- `POST /api/v1/export/dxf` — экспортировать проверенную геометрию;
- `GET /health` — проверка состояния.

## Проверка проекта

```bash
pip install -e '.[dev]'
ruff check .
pytest
```

Проект использует FastAPI, OpenCV и ezdxf. Ввод обрабатывается в памяти и не записывается на диск.
